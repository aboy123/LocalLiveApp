# 本地生活板块功能说明

## 🎯 功能概述

本地生活板块为用户提供周边商家信息查询服务,包括美食、购物、娱乐、生活服务等多个分类。

## ✨ 核心功能

### 1. 商家浏览
- **分类筛选**: 支持按2大分类浏览商家
  - 🧹 家政
  - 💬 聊天

- **搜索功能**: 支持按商家名称、描述、标签搜索
- **卡片展示**: 美观的商家卡片,显示评分、地址、标签等信息

### 2. 商家详情
- **详细信息**: 查看商家完整信息
  - 商家介绍
  - 地址信息
  - 营业时间
  - 价格信息
  - 联系方式
  
- **快捷操作**: 
  - 📞 一键拨打电话
  - ⭐ 收藏功能(预留)

### 3. 数据管理
- **本地存储**: 使用 localStorage 持久化商家数据
- **初始数据**: 预置6个示例商家
- **可扩展**: 支持动态添加新商家

## 🔗 访问入口

### 1. 首页入口
- 访问首页 http://localhost:5173/
- 点击 "🏙️ 本地生活" 按钮

### 2. 仪表板入口
- 登录后进入仪表板
- 点击 "🏙️ 本地生活" 卡片

### 3. 直接访问
- 本地生活列表: http://localhost:5173/local-life
- 商家详情: http://localhost:5173/local-life/:id

## 🎨 界面特色

### 本地生活主页
- **渐变头部**: 紫色渐变背景,带搜索框
- **分类导航**: 横向滚动分类按钮,选中高亮
- **商家网格**: 响应式网格布局,自适应屏幕
- **交互动效**: 卡片悬停上浮效果

### 商家详情页
- **大图展示**: 顶部商家图标展示
- **评分系统**: 星级评分 + 评价数量
- **标签云**: 彩色标签展示特色
- **信息分区**: 清晰的信息分组展示
- **固定底栏**: 拨打电话和收藏按钮

## 📊 数据结构

### 商家对象
```javascript
{
  id: number,              // 商家ID
  name: string,            // 商家名称
  category: string,        // 分类ID
  rating: number,          // 评分(0-5)
  reviews: number,         // 评价数量
  address: string,         // 地址
  phone: string,           // 电话
  description: string,     // 描述
  image: string,           // 图标(emoji)
  tags: array,             // 标签数组
  businessHours: string,   // 营业时间
  minPrice: number         // 人均消费
}
```

### 分类对象
```javascript
{
  id: string,              // 分类ID
  name: string,            // 分类名称
  icon: string,            // 图标(emoji)
  color: string            // 主题色
}
```

## 🔧 API 函数

### 数据获取
- `getMerchants()` - 获取所有商家
- `getMerchantsByCategory(categoryId)` - 按分类获取商家
- `getMerchantById(id)` - 获取商家详情
- `searchMerchants(keyword)` - 搜索商家

### 数据操作
- `addMerchant(merchant)` - 添加新商家

## 💡 使用示例

### 浏览全部商家
1. 进入本地生活页面
2. 默认显示所有商家
3. 点击任意卡片查看详情

### 按分类浏览
1. 点击顶部分类按钮
2. 只显示该分类的商家
3. 点击"全部"返回所有商家

### 搜索商家
1. 在搜索框输入关键词
2. 点击搜索按钮或回车
3. 显示匹配的商家列表

### 查看详情
1. 点击商家卡片
2. 进入详情页
3. 查看完整信息
4. 可拨打电话或返回
