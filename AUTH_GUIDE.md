# HTML5 App - 用户认证系统

这是一个基于 React + Vite 构建的现代化单页应用,包含完整的用户注册登录流程。

## 🚀 功能特性

### ✅ 已实现功能
- **用户注册**: 支持新用户注册,数据存储在本地
- **用户登录**: 邮箱密码登录验证
- **路由保护**: 未登录用户无法访问受保护的页面
- **会话保持**: 使用 localStorage 持久化用户状态
- **表单验证**: 完整的表单验证和错误提示
- **响应式设计**: 适配移动端和桌面端

```
src/

├── App.jsx                     # 主应用组件(路由配置)
└── main.jsx                    # 应用入口
```

## 🔐 认证流程

### 1. 注册流程
1. 用户访问 `/register` 页面
2. 填写姓名、邮箱、密码、确认密码
3. 前端验证(密码长度、两次密码一致性)
4. 检查邮箱是否已被注册
5. 保存用户数据到 localStorage
6. 自动登录并跳转到仪表板

### 2. 登录流程
1. 用户访问 `/login` 页面
2. 输入邮箱和密码
3. 验证用户凭据
4. 登录成功则保存用户状态
5. 跳转到仪表板

### 3. 路由保护
- `/dashboard` 需要登录才能访问
- 未登录用户会被重定向到登录页面
- 已登录用户可以正常访问

## 🛠️ 技术栈

- **React 18** - UI 框架
- **Vite 5** - 构建工具
- **React Router DOM** - 路由管理
- **Context API** - 状态管理
- **LocalStorage** - 数据持久化

## 📝 使用说明

### 启动开发服务器
```bash
cd html5-app
npm run dev
```

### 访问应用
打开浏览器访问: http://localhost:5173

### 测试流程
1. 访问首页,点击"注册"按钮
2. 填写注册表单完成注册
3. 自动跳转到仪表板
4. 点击"退出登录"返回登录状态
5. 尝试直接访问 `/dashboard`,会被重定向到登录页
